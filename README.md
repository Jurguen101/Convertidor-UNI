# Convertidor-UNI


🇪🇸 Español

Convertidor Personal de Descargas de Ingeniería
Desarrollado por: Br. Jurguen Adriel Delgadillo Jarquin | Universidad Nacional de Ingeniería (UNI)

Este programa es una herramienta de escritorio eficiente y segura, diseñada para descargar contenido de YouTube en la más alta calidad disponible. Construido íntegramente en Python, el software aprovecha la potencia de las bibliotecas yt-dlp y FFmpeg para extraer tanto video de alta resolución (MP4) como audio puro (MP3), eliminando la necesidad de depender de sitios web de terceros llenos de publicidad o límites de velocidad.

Características principales:

Interfaz Gráfica Intuitiva: Diseño moderno y con relieve construido con Tkinter.

Procesamiento Asíncrono: Las descargas se ejecutan en segundo plano, evitando que la aplicación se congele.

Monitoreo en Tiempo Real: Cuenta con una barra de progreso exacta que lee el flujo de datos en vivo.

Máxima Calidad: Fuerza la unión de las mejores pistas de video y audio disponibles en los servidores.

Ideal para estudiantes, docentes y profesionales que necesitan gestionar y respaldar recursos multimedia de forma rápida y sin complicaciones.

🇺🇸 English

Engineering Personal Download Converter
Developed by: Br. Jurguen Adriel Delgadillo Jarquin | Universidad Nacional de Ingeniería (UNI)

This program is an efficient and secure desktop tool designed to download YouTube content in the highest available quality. Built entirely in Python, the software leverages the power of the yt-dlp and FFmpeg libraries to extract both high-resolution video (MP4) and pure audio (MP3), eliminating the need to rely on ad-heavy third-party websites or speed-capped services.

Key Features:

Intuitive Graphical Interface: A modern, 3D-styled design built with Tkinter.

Asynchronous Processing: Downloads run in background threads, keeping the application responsive and preventing freezes.

Real-Time Monitoring: Features an accurate progress bar that reads the live data stream.

Maximum Quality: Forces the merging of the best available video and audio tracks from the servers.

Ideal for students, teachers, and professionals who need to manage and back up multimedia resources quickly and hassle-free.

🇪🇸 Guía de Instalación y Uso
📥 Cómo descargar e "instalar"
Este programa es portable, lo que significa que no necesitas instalar nada en tu sistema.

Ve a la sección de Releases (Lanzamientos) ubicada en la parte lateral derecha de este repositorio en GitHub.

Descarga el archivo comprimido más reciente (por ejemplo, Convertidor_Descargas_UNI.zip).

Extrae el contenido del archivo .zip en una carpeta de tu preferencia.

⚠️ IMPORTANTE: Asegúrate de no sacar el archivo ejecutable (.exe) principal de esa carpeta. Debe permanecer siempre junto a los archivos ffmpeg.exe y ffprobe.exe para poder procesar la alta calidad correctamente.

🚀 Cómo utilizar el programa

Haz doble clic en el archivo .exe para iniciar la aplicación.

Ve a YouTube, copia el enlace (URL) del video que deseas descargar y pégalo en el cuadro de texto del programa.

Selecciona el formato de salida deseado haciendo clic en los botones: Video MP4 o Audio MP3.

Haz clic en el botón verde "⬇️ INICIAR DESCARGA".

Se abrirá una ventana de tu sistema. Elige la carpeta donde quieres que se guarde el archivo final.

¡Listo! Relájate y observa la barra de progreso en tiempo real. La interfaz te avisará con un mensaje verde cuando tu archivo esté listo.

🇺🇸 Installation and Usage Guide
📥 How to download and "install"
This program is portable, meaning you don't need to install anything on your system.

Go to the Releases section located on the right sidebar of this GitHub repository.

Download the latest compressed file (e.g., Convertidor_Descargas_UNI.zip).

Extract the contents of the .zip file into a folder of your choice.

⚠️ IMPORTANT: Make sure not to move the main executable (.exe) file out of that folder. It must always stay alongside the ffmpeg.exe and ffprobe.exe files to process high quality properly.

🚀 How to use the program

Double-click the .exe file to launch the application.

Go to YouTube, copy the link (URL) of the video you want to download, and paste it into the program's text box.

Select your desired output format by clicking the buttons: Video MP4 or Audio MP3.

Click the green "⬇️ INICIAR DESCARGA" (Start Download) button.

A system window will pop up. Choose the destination folder where you want to save the final file.

That's it! Sit back and watch the real-time progress bar. The interface will notify you with a green message when your file is ready.
